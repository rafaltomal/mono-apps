# Mono Apps

Simple productivity tools that respect your privacy.

![Mono Apps Dashboard](https://img.shields.io/badge/vanilla-HTML%2FCSS%2FJS-blue) ![License](https://img.shields.io/badge/license-MIT-green)

## Features

- **Free to use** - No registration required
- **Privacy-first** - All data saved locally in your browser
- **Works offline** - No internet connection needed
- **No tracking** - Zero analytics or data collection

## Apps

### Focus Flow
Distribute tasks across days. Enter your tasks, set how many you want to complete per day, choose your working days, and let Focus Flow create a schedule for you.

### Focus Timer
Pomodoro-style focus sessions. Work for 25 minutes, take a 5-minute break. Tracks your daily sessions and total focus time.

### Daily Tasks
Simple daily to-do list. Add tasks, check them off, and start fresh each day.

### Habit Tracker
Build consistent habits. Track your daily habits with a visual calendar showing your streaks and progress.

### Workout Tracker
Plan your workouts. Create exercises with sets, reps, and optional timers. Track your training sessions.

## Getting Started

### Option 1: Use Online
Visit the hosted version at https://rafaltomal.com/mono-apps

### Option 2: Run Locally
1. Clone this repository:
   ```bash
   git clone https://github.com/rafaltomal/mono-apps.git
   ```
2. Open `index.html` in your browser

No build process, no dependencies, no setup required.

## Data Management

Your data is stored in your browser's localStorage. Use the **Export Data** and **Import Data** buttons on the dashboard to backup or transfer your data between devices.

## Tech Stack

- Vanilla HTML, CSS, and JavaScript
- ES6 Modules
- CSS Custom Properties (variables)
- localStorage for persistence
- No frameworks or build tools

## Mobile Usage

For the best experience on mobile devices, add Mono Apps to your home screen:

### iPhone / iPad
1. Open https://rafaltomal.com/mono-apps in Safari (or your own hosted version)
2. Tap the Share button (square with arrow)
3. Scroll down and tap **Add to Home Screen**
4. Name it "Mono Apps" and tap **Add**

The app will appear on your home screen and launch like a native app.

### Android
1. Open https://rafaltomal.com/mono-apps in Chrome (or your own hosted version)
2. Tap the three-dot menu (â‹®)
3. Tap **Add to Home screen** (or **Install app** if available)
4. Confirm by tapping **Add**

Android may also prompt you to install the app automatically when visiting the site.
